# Computer Animation

Course description, tbd.

# Goals/Competencies

After the completion of the course it is expected for students to be capable of using basic methods for productive creation of computer based animations and working with applications for creating animations. They should be able to model, apply textures, rig, add lights, animate, render and compose digital scenes.

# Course Content

  • basic principles of 3D space
  • describing digital scenes
  • hierarchical organization
  • polygonal geometry
  • modeling of digital objects and characters
  • NURBS modeling
  • deformation techniques
  • rigging of objects and characters
  • principles of animation
  • animating digital characters
  • shading and materials
  • texturing
  • UV coordinates
  • lights in digital scenes
  • cameras
  • scripting and automation
  • compositing and post-production

# Learning Methods

Lectures using presentations, interactive lectures, exercises (using equipment and software packages), teamwork, case studies, invited guest lecturers, independent preparation and defense of a project assignment and seminar work.
